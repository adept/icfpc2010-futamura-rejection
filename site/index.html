New team members guide

  1. Join our Jabber conference: futamura@conference.jabber.ru (channel password will be given to you by infrastructure guy, Alexander Temerev);
  2. Clone our Git repository, ssh://pubmath.org/git/icfp.git (account and password should be given to you by infrastructure guy again);
  3. Download and install TeamSpeak 3 client and connect to our TS server (ts.pubmath.org); join 'Futamura' channel for voice communications;
  4. Configure your TeamSpeak client for push-to-talk communication;
  5. Wait for the contest task being posted, then we will discuss it in Jabber and TeamSpeak.
